# Key reference

## Tabs

| Key               | Description   |
| ----------------- | ------------- |
| `^b ^c`           | New tab       |
| `^b ^x`           | Close tab     |
| `^b ^n` / `^b `p` | Next/prev tab |
| `^b ,`            | Rename tab    |
| `^b {0..9}`       | Switch to tab |
| `^b ^w`           | Switch to...  |

## Panes

| Key                       | Description          |
| ------------------------- | -------------------- |
| `^b ^s` (default: `^b "`) | Split horiz          |
| `^b ^v` (def: `^b %`)     | Split vert           |
| `^b o`                    | Focus next pane      |
| `^b q`                    | Focus pane...        |
| `^b {`                    | Move pane            |
| `^b !`                    | Move pane to new tab |

## Sessions

| Key    | Description    |
| ------ | -------------- |
| `^b #` | New session    |
| `^b $` | Rename session |
| `^b *` | Close session  |

## Other shortcuts

| Key    | Description                 |
| ------ | --------------------------- |
| `^b ?` | Help                        |
| `^b b` | Toggle status bar           |
| `^b I` | Install plugins             |
| `^b R` | Reload config               |

## Copying

| Key    | Description                 |
| ------ | --------------------------- |
| `^b [` | Scroll mode (exit with `q`) |

`space` to select in scroll mode, then `Enter` to copy to clipboard
